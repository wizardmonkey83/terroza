A Note from the Creator

Terroza is my first major solo project, created due to my desire to build a complete, full-stack application.

Due to my prior knowledge of python, I decided to build Terroza using Django for the backend and HTMX for the frontend. Django's "batteries-included" framework enabled me to build a secure, robust backend efficiently and on the frontend side, HTMX allowed me to create a single-page-app feel without needing to learn JavaScript frameworks, yet.

Please reach out if you encounter bugs or have suggestions for the site, after all it's the first iteration.

Thank you for visiting Terroza. I hope you enjoy using it as much as I enjoyed building it.
